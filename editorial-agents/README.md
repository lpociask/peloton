# Agenci redakcyjni PAPER

Ten katalog definiuje dziesięć fikcyjnych person pisarskich: pięć dla **Pelotonu** i pięć dla **Rowków**. Każdy agent odpowiada za jeden stały obszar i jeden z pięciu materiałów numeru.

Redaktor naczelny obu magazynów pozostaje człowiekiem, podejmuje decyzje wydawnicze i jako jedyny pisze wstępniaki. Agenci nie są autonomicznymi wydawcami. Przygotowują udokumentowane szkice do redakcji i akceptacji.

## Zasada użycia

1. Wczytaj [`CONTRACT.md`](./CONTRACT.md).
2. Wczytaj plik wybranej postaci.
3. Przekaż brief redaktora naczelnego, materiały źródłowe i termin weryfikacji danych.
4. Najpierw odbierz mapę źródeł oraz rejestr tez, dopiero potem zamów pełny szkic.

Specyfikacja i mapowanie wygenerowanych portretów znajduje się w [`PORTRAIT_PROMPTS.md`](./PORTRAIT_PROMPTS.md).
